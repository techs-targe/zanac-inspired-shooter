## 🛠️ トラブルシューティング

### Q: ページが表示されない
```
A:
1. URLが正しいか確認: http://localhost:8000
2. サーバーが起動しているか確認
3. 他のアプリケーションがポート8000を使用していないか確認
```

### Q: 古いバージョンが表示される
```
A: キャッシュの問題です。以下を試してください：

【最優先】シークレットモードで開く
  Chrome: Ctrl + Shift + N
  Firefox: Ctrl + Shift + P

【次に試す】強制リロード
  Ctrl + Shift + R （Windows/Linux）
  Cmd + Shift + R （Mac）

【それでもダメなら】開発者ツールでキャッシュ無効化
  1. F12 を押す
  2. Network タブを開く
  3. "Disable cache" にチェック ✅
  4. Ctrl + Shift + R でリロード
```

### Q: 変更が反映されない
```
A:
1. 開発者ツール（F12）を開く
2. Network タブをクリック
3. bullet.js をクリック
4. Headers タブで Request URL を確認
5. "t=1731251200" が含まれているか確認
```
